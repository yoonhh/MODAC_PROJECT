<h2> 개요 및 기획의도 </h2>
<li>코로나 때문에 사람이 모여있는 여행지를 갈 수 없다. -> 탁트인 공간의 캠핑이 유행!</li>
<li>하지만 캠핑장에 대한 정보는 여기저기에 흩어져있고 비교하기 쉽지 않다.</li>
<li>캠핑장과 캠핑에대한 검색을 용이하게 하고 캠퍼들끼리 정보를 쉽게 공유할 수 있는 커뮤니티 사이트를 고안하였습니다.</li>
